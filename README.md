# 🍣 Sushi Counter

Live Demo:
https://luzzo17.github.io/counter/

# Descrizione

Sushi Counter è un semplice progetto realizzato in JavaScript che permette di tenere traccia del numero di sushi consumati tramite un sistema di incremento e decremento.

L'interfaccia viene generata dinamicamente attraverso la manipolazione del DOM: tutti gli elementi principali (counter, bottoni, immagini, footer) vengono creati e gestiti esclusivamente tramite JavaScript.

Il progetto salva automaticamente il valore del contatore nel localStorage, in modo che il numero venga mantenuto anche dopo il refresh della pagina.

# Funzionalità

Incremento del counter (+1, +2, +3, +4, +8, a seconda della portata)

Decremento del counter (-1, -2, -3, -4, -8)

Blocco del valore minimo a 0 (non può andare in negativo)

Cambio dinamico dell'immagine in base all'azione

Reset del counter

Persistenza dei dati tramite localStorage

Layout responsive per dispositivi mobili
